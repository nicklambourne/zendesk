# Zendesk Coding Challenge 2019

## Requirements
- Python 3.6
- virtualenv

## Installing
### Unix
`python3 -m venv venv`

`source venv/bin/activate`

`python setup.py install`

`python -m zendesk run`

### Windows
`virtualenv venv`

`venv\Scripts\activate`

`python setup.py install`

`python -m zendesk run`
